项目介绍
开发工具
前端:cursor、hbuiderx、微信开发者工具
后端:idea、navicat、mysql
脚手架:字母哥助手
架构
前后端分离架构,采用SpringBoot3+Vue3+Uniapp来实现网页端和移动端
功能调研
健康数据记录与追踪
能记录用户的各类信息包括身体指标和健康指标,比如心率、体重、身高、血糖、血脂等等
可视化的数据分析
接入AI来实现个人数据的分析汇总 (DeepSeek大模型)（创新）
饮食管理
提供一个完善的饮食资料库，具体包括到食物的重量、热量、蛋白质、脂肪、有机盐等等
提供使用者可以进行饮食记录(选择记录,减少用户的输入)
基于用户通过微信小程序语音输入自动识别用户的饮食记录（创新）
运动管理
提供一个完善的运动选项，具体包括到跑步、游泳、登山、打羽毛球、打篮球等等
提供使用者可以进行运动记录(选择记录,减少用户的输入)
基于用户通过微信小程序语音输入自动识别用户的运动记录（创新）
健康提醒和目标设定
支持用户指定计划,设定提醒次数和提醒时间
提醒的方式(短信、邮箱、微信通知)
健康知识和食谱大全
用户可以浏览健康知识和食谱大全
支持评价、点赞、收藏
根据用户的喜好进行推荐（推荐算法）
